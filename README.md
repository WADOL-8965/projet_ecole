#  Analyse des performances scolaires

##  Objectif
Ce projet a pour objectif d’analyser les performances des élèves à partir de leurs notes.  
Il permet de calculer les moyennes par élève et par classe, d’identifier les meilleurs élèves et de visualiser les résultats.

##  Données
Les données proviennent d’un fichier CSV (`Ecole.csv`) contenant :  
- Nom des élèves  
- Classe  
- Matières  
- Notes  

## Analyse
- Calcul de la moyenne par élève  
- Identification du top 5 des meilleurs élèves  
- Détermination du meilleur élève  
- Calcul de la moyenne des notes par classe  
- Analyse descriptive des données  

##  Résultats
- Classement des élèves selon leur moyenne  
- Meilleur élève identifié avec sa moyenne  
- Classement des classes selon leurs performances  
- Visualisation des résultats avec plusieurs graphiques  

##  Outils
- Python  
- Pandas  
- Matplotlib  
- Seaborn
